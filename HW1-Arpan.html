import warnings
import numpy as np
import pandas as pd
from statsmodels.tsa.stattools import adfuller, kpss
warnings.filterwarnings("ignore", category=pd.errors.PerformanceWarning)
warnings.filterwarnings("ignore", category=pd.errors.SettingWithCopyWarning)

def stationarity_tests(time_series):
    # ADF Test
    adf_result = adfuller(time_series)
    adf_output = {
        'Test Statistic': adf_result[0],
        'p-value': adf_result[1],
        'Lags Used': adf_result[2],
        'Number of Observations Used': adf_result[3],
        'Critical Values': adf_result[4],
        'ADF Test Result': 'Stationary' if adf_result[1] < 0.20 else 'Non-Stationary'
    }
    
    # KPSS Test
    kpss_result = kpss(time_series, regression='c', nlags="auto")
    kpss_output = {
        'Test Statistic': kpss_result[0],
        'p-value': kpss_result[1],
        'Lags Used': kpss_result[2],
        'Critical Values': kpss_result[3],
        'KPSS Test Result': 'Non-Stationary' if kpss_result[1] < 0.05 else 'Stationary'
    }
    
    return {
        'ADF Test': adf_output,
        'KPSS Test': kpss_output
    }

import pandas as pd
from statsmodels.tsa.stattools import adfuller, kpss

def difference_and_stationarity_tests(series):
    # Compute first differences
    diff_series = series.diff().dropna()  # Drop NA resulting from differencing
    
    # Performing ADF test
    adf_result = adfuller(diff_series)
    adf_statistic = adf_result[0]
    adf_p_value = adf_result[1]
    
    # Performing KPSS test
    kpss_result = kpss(diff_series, regression='c', nlags="auto")
    kpss_statistic = kpss_result[0]
    kpss_p_value = kpss_result[1]
    
    # Returning results
    return {
        'ADF Statistic': adf_statistic,
        'ADF p-value': adf_p_value,
        'KPSS Statistic': kpss_statistic,
        'KPSS p-value': kpss_p_value
    }

# Applying the function to each time series

results_spy = difference_and_stationarity_tests(data['SPY'])

F. In general, explain how you can get different stationarity results from the two tests.

The ADF test: Presence of a unit root in the time series, which indicates non-stationarity. It tests whether a series has a stochastic trend that can be removed through differencing. A lower ADF test statistic and a smaller p-value suggestnon-stationary. In contrast, the KPSS test examines the null hypothesis that the series is stationary around a deterministic trend, such as a linear trend or polynomial trend. It measures the deviation from this stationary process, with a higher KPSS statistic and a smaller p-value indicating stronger evidence against stationarity. Therefore, differences in the stationarity results from these tests can arise due to their differing null hypotheses, sensitivity to different types of trends, and the underlying assumptions about the structure of the time series data.

Ans: We can use ADF and KPSS test to determine whether a random walk is stationary. Both the ADF and KPSS tests provide evidence that the random walk series is non-stationary:

The ADF test suggests the presence of a unit root (non-stationarity) due to the non-rejection of Null hypothesis. The KPSS test confirms non-trend-stationarity due to the rejection of Null hypothesis.

Therefore, based on these quantitative tests, we conclude that the random walk is not stationary.They exhibit trends and movements that are not stationary over time.
